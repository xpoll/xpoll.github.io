jekyll 是目前我见到的最满意的博客

{% highlight ruby %}
gem install jekyll bundle
jekyll -v
jekyll new myblog
cd myblog
jekyll server
#=> browser 'http://127.0.0.1:4000'.
{% endhighlight %}
